# Connecting with SOC technologies

This is a tutorial of connecting SIEMs and SOARs to the platform.

## Supported Platforms

**SOAR:** [The Hive](https://strangebee.com/)

**SIEM:** [Wazuh](https://wazuh.com/)


## Setup Instructions

First setup and run the server. Refer to [this guide](/documents/docs/get_started) for setup instructions. If you have finished setup, run the server with:

```bash
python manage.py runserver 0.0.0.0:8000
```

The settings page can be located in the hamburger menu at the top left:
